# Sequential Thinking MCP

Use this app when the user wants structured reasoning, step-by-step problem decomposition, or a revisable plan for a complex task.

- This app exposes tools from the Sequential Thinking MCP server.
- The app uses a stdio MCP endpoint and does not require authentication.
- Prefer the MCP tools when the task benefits from explicit intermediate thoughts, revisions, branching, or tracking how many thinking steps remain.
- Do not use it for simple one-step questions where a direct answer is enough.
